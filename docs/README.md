# Материалы к диплому «Автоматизация бухгалтерской деятельности в сфере социальных услуг»

Эта папка содержит конспекты, анализ и вспомогательные материалы для подготовки к защите выпускной квалификационной работы.

## Файлы и назначение

| Файл | Назначение | Связь с дипломом |
|------|------------|------------------|
| `диплом.txt` | Исходный текст диплома | Полный текст работы |
| `chapter1_summary.md` | Конспект главы 1 — Аналитическая часть | Страницы 6–41 диплома |
| `chapter2_summary.md` | Конспект главы 2 — Проектная часть | Страницы 42–68 диплома |
| `chapter3_summary.md` | Конспект главы 3 — Экономическая часть | Страницы 69–82 диплома |
| `intro_conclusion_summary.md` | Конспект введения и заключения | Страницы 3–5, 82–85 диплома |
| `defense_questions.md` | Вопросы комиссии и ответы к защите | Общий охват работы |
| `defense_plan.md` | План защиты диплома | Структура выступления, тайминг, опасные вопросы |
| `defense_script.md` | Текст выступления | Готовый текст на 7–9 минут |
| `review_md_files.md` | Отчёт о проверке md-файлов | Методический разбор |
| `diploma_issues.md` | Реестр выявленных ошибок диплома | Актуальные правки |
| `concrete_edits.md` | Конкретные правки по главам | Редакционные указания для внесения в диплом |

## Как пользоваться

1. **Для повторения содержания** — читать `chapter1_summary.md`, `chapter2_summary.md`, `chapter3_summary.md`, `intro_conclusion_summary.md`.
2. **Для подготовки к выступлению** — использовать `defense_plan.md` и `defense_script.md`.
3. **Для подготовки к вопросам** — использовать `defense_questions.md`.
4. **Для понимания слабых мест** — изучить `diploma_issues.md` и разделы «Слабые стороны» в каждом конспекте.
5. **Для проверки качества конспектов** — открыть `review_md_files.md`.

## Обозначения в файлах

- `[диплом, с. X]` — факт из исходного текста диплома.
- `[рекомендация]` — предложение по улучшению, не содержащееся в дипломе.
- `[ошибка в дипломе]` — выявленная неточность в исходном тексте.
- `[требует уточнения]` — противоречивое или неясное место.

## Правила работы с материалами

- Все цифры и формулы должны соответствовать диплому, если не указано иное.
- Перед защитой рекомендуется сверить конспекты с финальной версией диплома.
- Вопросы в `defense_questions.md` разделены на «по тексту диплома» и «возможные дополнительные».
